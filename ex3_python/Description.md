# dirtree.py
For start script you need
``` bash
python3 dirtree.py (-r) <any files and folders>
# For example
python3 dirtree.py
python3 dirtree.py -r #(for recursive check current directory)
python3 dirtree.py folder1 #(for check other directory)
python3 dirtree.py -r folder2 #(for recursive check other directory)
```

# md5sum.py
For start script you need
``` bash
python3 md5sum.py (-r) <any files and folders>
# For example
python3 md5sum.py
python3 md5sum.py -r #(for recursive check current directory)
python3 md5sum.py folder1 folder2 file1 file2 #(for checr this directories and files)
python3 md5sum.py -r folder1 folder2 file1 file2 #(for recursive checr this directories and files)
```
